The Midnight Sky UI

Introduction

Midnight Sky UI is a collection of CSS styles and UI components to make websites look seamlessly beautiful. Inspired by the Starry night painting by Vincent Van gogh.

Quick Start

To quickly add Midnight Sky UI to your project:
Copy-paste the stylesheet <link> into your <head> before all other stylesheets

<link rel="stylesheet" href="https://the-midnight-sky-ui.netlify.app/midnightsky-ui.css">

Starter template

Be sure to have your pages set up with the latest design and development standards. That means using an HTML5 doctype and including a viewport meta tag for proper responsive behaviors. Put it all together and your pages should look like this:

<!doctype html>
<html lang="en">
<head>
<!-- Required meta tags -->
<meta charset="utf-8">
<meta name="viewport" content="width=device-width, initial-scale=1">

<!-- Midnight Sky CSS -->
<link href="https://the-midnight-sky-ui.netlify.app/midnightsky-ui.css" rel="stylesheet">

<title>Hello, world!</title>
</head>
<body>
<h1>Hello, world!</h1>
</body>
</html>

Rating

To use stars for rating, create a div with class "stars". And then pass the rating value as an inlined custom CSS property (--rating). This means there is no additional rendering logic required, except for displaying the same rating value in the label for better accessibility.

<div class="stars" style="--rating: 2.5;" aria-label="Rating of this product is 2.5 out of 5."></div>
